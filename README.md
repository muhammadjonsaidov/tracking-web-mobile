# Tracking App (React + Vite)

Oddiy React ilovasi uchun minimal loyiha tuzilmasi.

## Ishga tushirish

1. Kutubxonalarni o‘rnating:
   npm install
2. Dasturchi rejimida ishga tushiring:
   npm run dev
3. Ishlab chiqarish build:
   npm run build

## Qisqacha tuzilma

- `src/` — asosiy kod
- `src/app/` — ilova routing va layout
- `src/features/` — funksional modullar
- `src/shared/` — umumiy UI, hook va util fayllar

## Talablar

- Node.js 18+ (tavsiya)
